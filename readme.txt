1. Сверстайте HTML-страницу.
Сверстайте CSS-документ стилей.

2. Назначьте класс заголовку
Назначьте класс параграфу с датой
Назначьте класс параграфу с основным текстом
Назначьте класс параграфу с подписью

3. Стилизуйте список при помощи псевдоклассов
Создайте маркированный список из пяти имён: John, Sam, Mary, Tom, Jane.
Используя псевдоклассы, опишите стили элементов в соответствии с правилами:
- цвет текста первого элемента — blue;
- цвет текста второго элемента — orange;
- цвет текста последнего элемента — red;
- при наведении, любой элемент списка меняет цвет текста на green.

Цвет фона каждого элемента списка сделайте соответствующим его содержанию.
Следите за тем, чтобы текст читался. При необходимости меняйте цвет текста.

5. Стилизуйте веб-страницу в соответствии с требованиями.
